Public surfaces used

This app uses public, server-rendered pages and the JSON already embedded in those pages. It does not call login, edit, delete, report, or protected phone reveal endpoints.

Listings{{.BaseURL}}/en/ads and category/location variants like /en/ads/colombo/mobile-phones.
Search/en/ads?query=... for user-entered keyword searches.
Details{{.BaseURL}}/en/ad/{slug} for public ad detail, images, seller name, and phone numbers present in page data.
Imageshttps://i.ikman-st.com/{slug}/{image-id}/{width}/{height}/cropped.jpg and fitted.jpg.
API root observedThe page sets window.apiURL = "https://api.ikman.lk", but this app avoids private/authenticated API calls.

Respectful-use guardrails: the client has a timeout, in-memory cache, and outbound request interval. Use it for user-driven browsing, not bulk harvesting.
